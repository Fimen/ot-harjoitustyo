## Komentorivitoiminnot

### Testaus

Testit suoritetaan komennolla

```
mvn test
```

Testikattavuusraportti luodaan komennolla

```
mvn jacoco:report
```

### Suoritettavan jarin generointi

Komento

```
mvn package
```

### JavaDoc

JavaDoc generoidaan komennolla

```
mvn javadoc:javadoc
```
